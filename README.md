# vue-boilerplate
## Synopsis
Boilerplate for a Vue app.

## Usage
To get started:
```
  git clone https://github.com/joncass/vue-boilerplate
  yarn
  yarn dev
```
then visit `localhost:4000` in your browser.

Other useful scripts:
- `yarn lint` - Run the linter.
